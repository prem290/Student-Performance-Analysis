# Student Performance Analysis

## Objective
Analyze student performance and identify factors affecting academic success.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Data Exploration
- Feature Engineering
- Factor Analysis
- At-Risk Student Segmentation

## Visualizations
- Box Plot
- Bar Chart
- Correlation Heatmap
- Grouped Bar Chart
- Histogram
- Scatter Plot

## Results
- Higher parental education is associated with better student performance.
- Test preparation significantly improves scores.
- Reading and Writing scores are strongly correlated.
- Female students perform better in language subjects.
- At-risk students were successfully identified for targeted intervention.
